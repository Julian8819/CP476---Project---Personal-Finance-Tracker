# Personal Expense Tracker

## Project Overview
Personal Expense Tracker is a full-stack web application developed as a group project for  
**CP476A â€“ Internet Computing**.

The application allows a user to record income and expenses, edit or delete transactions,
and view a financial summary of their data.  
The system is designed for a single user and does not require user accounts or authentication.

The project is being developed in milestones, beginning with planning and design, followed by
implementation and final delivery.

## Project Objectives
- Allow users to track income and expenses
- Support full CRUD operations on financial transactions
- Provide a financial summary of income versus expenses
- Demonstrate server-side processing and relational database usage
- Apply data validation and basic security practices

## Core Features (Planned)
- Add income transactions
- Add expense transactions
- Edit existing transactions
- Delete transactions
- View financial summary (total income, total expenses, net balance)
